# Mohammed Riyas - Personal Portfolio

Welcome to my personal portfolio website! This is a responsive frontend portfolio showcasing my skills, projects, and experience as a frontend developer.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Sections Included**:
  - Home: Featured introduction with animated typing effect
  - About: Personal introduction and technical skills
  - Projects: Showcase of completed projects and experience
  - Contact: Contact form with email integration using EmailJS
- **Smooth Scrolling**: Elegant scroll reveal animations
- **Mobile Menu**: Responsive hamburger menu for mobile devices
- **Contact Form**: Functional contact form with email validation

## Features Details

### Typing Animation
The home section features a typing animation that cycles between "Designer" and "Developer" using Typed.js library.

### Scroll Animations
Elements animate into view as you scroll down the page using ScrollReveal.js.

### Contact Form
The contact form uses EmailJS to send messages directly to your email without requiring a backend server. You'll need to:
- Sign up on [EmailJS](https://www.emailjs.com/)
- Update the service ID, template ID, and public key in `main.js`

### Responsive Design
The portfolio is fully responsive with breakpoints at:
- 1024px (tablets)
- 900px (small devices)
- 540px (mobile phones)
